# 腾讯云域名注册步骤

## 一、注册腾讯云账号

### 步骤 1: 访问腾讯云官网

访问：https://cloud.tencent.com/

### 步骤 2: 注册/登录账号

1. 如果已有账号，直接登录
2. 如果没有账号，点击"注册"
3. 填写手机号、验证码、设置密码
4. 完成实名认证（需要身份证）

---

## 二、购买域名

### 步骤 1: 进入域名注册页面

1. 登录腾讯云控制台
2. 在顶部搜索框输入"域名注册"或访问：https://dnspod.cloud.tencent.com/
3. 点击"域名注册"

### 步骤 2: 搜索域名

1. 在搜索框输入你想要的域名（不含后缀）
   - 例如：输入 `furigana-app` 或 `nihongo-tool`
2. 选择后缀（推荐 `.com` 或 `.cn`）
3. 点击"查询"

### 步骤 3: 选择域名

1. 查看查询结果
   - ✅ **可注册**：显示价格，可以购买
   - ❌ **已注册**：显示"已注册"，需要换一个
2. 点击"加入购物车"或"立即购买"

### 步骤 4: 确认订单

1. 选择注册年限（通常首年有优惠）
2. 确认价格
3. 选择"域名信息模板"（如果没有，需要先创建，见下方）
4. 点击"提交订单"

### 步骤 5: 支付

1. 选择支付方式（微信/支付宝/银行卡）
2. 完成支付
3. 等待域名注册成功（通常几分钟内）

---

## 三、创建域名信息模板（首次注册需要）

### 步骤 1: 进入域名控制台

1. 登录腾讯云控制台
2. 进入"域名注册" → "信息模板"

### 步骤 2: 创建模板

1. 点击"创建信息模板"
2. 填写信息：
   - **域名所有者类型**：个人 或 企业
   - **域名所有者**：你的姓名（个人）或公司名称（企业）
   - **联系人**：你的姓名
   - **联系电话**：手机号
   - **邮箱**：常用邮箱
   - **地址**：详细地址
   - **证件类型**：身份证（个人）或营业执照（企业）
   - **证件号码**：身份证号或营业执照号
3. 上传证件照片（身份证正反面或营业执照）
4. 点击"提交"

### 步骤 3: 等待审核

- 通常 1-3 个工作日审核完成
- 审核通过后，模板状态显示"已实名认证"

---

## 四、配置域名解析

### 步骤 1: 进入 DNS 解析

1. 登录腾讯云控制台
2. 进入"域名注册" → "我的域名"
3. 找到你刚注册的域名
4. 点击"解析"

### 步骤 2: 添加 A 记录

需要**添加两条 A 记录**，分别对应根域名和 www 子域名。

**第一条记录（根域名）：**

1. 点击"添加记录"
2. 填写：
   - **主机记录**：`@`（表示根域名，访问 `nihongo-tool.cn` 时生效）
   - **记录类型**：`A`
   - **线路类型**：默认（或"全网默认"）
   - **记录值**：`43.135.112.39`
   - **TTL**：600
3. 点击"保存"

**第二条记录（www 子域名）：**

1. 再次点击"添加记录"
2. 填写：
   - **主机记录**：`www`（表示 www 子域名，访问 `www.nihongo-tool.cn` 时生效）
   - **记录类型**：`A`
   - **线路类型**：默认（或"全网默认"）
   - **记录值**：`43.135.112.39`
   - **TTL**：600
3. 点击"保存"

完成后，解析列表里会有两条 A 记录：`@` 和 `www`，记录值都是你的服务器 IP。

### 步骤 3: 验证解析

等待 5-30 分钟，然后验证：

```bash
# 在命令行执行（Windows: cmd 或 PowerShell，Mac/Linux: Terminal）
nslookup yourdomain.com

# 应该返回你的服务器 IP: 43.135.112.39
```

或在浏览器访问：http://yourdomain.com:3001（应该能访问到你的网站）

## 五、常见问题

### Q1: 域名注册后多久生效？

**答**：通常几分钟内生效，最长不超过 24 小时。

### Q2: 域名解析多久生效？

**答**：
- 国内：通常 5-30 分钟
- 国外：可能需要 1-2 小时
- 最长不超过 48 小时

### Q3: 如何知道域名解析是否生效？

**答**：
```bash
# Windows (cmd)
nslookup yourdomain.com

# Mac/Linux
dig yourdomain.com
# 或
nslookup yourdomain.com
```

如果返回 `43.135.112.39`，说明解析已生效。

### Q4: 域名注册后需要备案吗？

**答**：
- **使用 `.cn` 域名**：如果服务器在国内，需要备案
- **使用 `.com` 等国际域名**：
  - 服务器在国内：需要备案
  - 服务器在国外：不需要备案
- **当前情况**：你的服务器在腾讯云（国内），建议备案

### Q5: 域名备案流程

**答**：
1. 登录腾讯云控制台
2. 进入"网站备案"
3. 填写备案信息（域名、服务器信息等）
4. 上传资料（身份证、网站信息等）
5. 等待审核（通常 7-20 个工作日）

**注意**：备案期间域名可以正常使用，但建议尽快完成备案。

### Q6: 域名注册后可以修改信息吗？

**答**：
- **域名所有者**：不能修改（除非过户）
- **DNS 解析**：可以随时修改
- **联系人信息**：可以修改

### Q7: 域名到期后怎么办？

**答**：
- 域名到期前会收到邮件提醒
- 到期后 30 天内可以续费
- 超过 30 天可能被他人注册

**建议**：设置自动续费，避免域名丢失。

## 六、域名选择建议

### 推荐命名规则

1. **简短易记**：6-12 个字符
2. **包含关键词**：
   - `furigana-xxx.com`
   - `nihongo-xxx.com`
   - `japanese-xxx.com`
3. **避免**：
   - 数字和连字符过多
   - 容易拼错的单词
   - 商标冲突

### 推荐后缀

1. **`.com`**（首选）
   - 价格：约 83 元/首年
   - 全球认知度最高
   - 适合长期品牌建设

2. **`.cn`**（国内用户）
   - 价格：约 33 元/首年
   - 国内用户熟悉
   - 需要实名认证
